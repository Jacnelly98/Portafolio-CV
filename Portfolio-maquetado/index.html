¿Qué te ofrezco?

Diseño de Interfaces

Diseñar interfaces intuitivas y atractivas con una jerarquía visual clara, coherencia estética y accesibilidad para optimizar la experiencia de usuario.

Diseño de Experiencia de Usuario

Crear interacciones significativas entre usuarios y productos, para lograr objetivos específicos a través de una experiencia fácil, eficiente y satisfactoria.

Diseño Gráfico

Diseñar gráficos visualmente impactantes y coherentes, utilizando elementos de diseño como tipografía, color, composición y marca para comunicar de manera efectiva.

Fotografía

Capturar momentos únicos y emociones a través de imágenes creativas y estéticamente agradables, utilizando técnicas de iluminación, composición y edición para transmitir una historia visual.

Maquetado de Páginas Web

Crear diseños de sitios web atractivos y funcionales, utilizando lenguajes de programación como HTML y CSS para garantizar una estructura sólida, una experiencia de usuario intuitiva y una estética coherente.

Proyectos centrados en el Usuario

Desarrollar soluciones innovadoras y efectivas, a través de un enfoque centrado en el usuario, que comprenda y responda a las necesidades, deseos y expectativas de los usuarios finales para mejorar su experiencia y satisfacción.
